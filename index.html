Projects

Snap Receipts

A web app for receipt tracking and digitizing your receipts. The app reads text data from an uploaded image using the Tesseract OCR Library. User interface created with AJAX and jQuery. Receipt location is geocoded and tracked with Google Maps

JuiceBox Ideaz

A collaboration site for users and teams to brainstorm, share and pursue ideas. Rspec driven TDD user authentication. User interface integrated with markdown for user specified styling and jQuery UI. Ideas can be dynamically edited.

CodeCore Alumni

Site for reconnecting with fellow graduates. User authentication created with bcrypt with OAuth log in support using Slack. Website was designed with cross platform capability in mind, following responsive design patterns.

PM Tool

The Project Management Tool is a web application that helps users collaborate on projects online as well as delegate tasks. Landing page features parallax scrolling. User authentication integrated with Devise. Page layout and styling done with Bootstrap and custom css. User interface for creating tasks, discussions and comments are added using AJAX.

codecoreQ

A joint group project where we created an interactive quiz site for measuring individual skill levels in coding. Frontend developed with HTML, SASS and jQuery. Backend developed using Ruby on Rails and PostgreSQL. User is provided feedback for answers, and can continue when answered correctly.

Launch Igniter

A crowd funding Rails web app. Created using TDD with Rspec tests. User authentication was built from scratch using bcrpyt. API integration for Fundsy Client with AJAX requests. Payment/Donation functionality using Stripe. Geocoding support for Google Maps.
