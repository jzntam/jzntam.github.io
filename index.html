Snap Receipts

A web app created for receipt tracking and digitizing your receipts. The app reads text data from an uploaded image using the Tesseract OCR Library. Website was designed with cross platform capability in mind, following responsive design patterns. User interface created with AJAX and jQuery. File uploads handled with CarrierWave. While receipt location is geocoded and tracked with Google Maps
